# ■ UI/Fonts の内容
* フォント関連のファイルが置かれている。
	* Font
		* Font アセット。
		* Font Face を複数登録できる。
		* UI Widget はこれを参照する。
	* Font Face
		* Font Face アセット。
		* otf/ttf をインポートすると作られる。
		* フォントデータの本体。
